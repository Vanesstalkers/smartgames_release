# domain/user

Доменная логика пользователя для release: тонкая обёртка над пользователем игры.

## Структура

- **Class.js** — класс `ReleaseUser`: наследует `domain.game.User` без добавления полей или методов. Используется для подстановки доменного пользователя в контексте release.

- **Session.js** — переопределение сессии пользователя для домена release (специфика сессии при необходимости).

## Связи

- Наследует `domain.game.User`, который в свою очередь расширяет контекст пользователя в игре (`lib.game.User`).
- Подключение класса в приложении задаётся через конфигурацию/регистрацию домена (impress подставляет domain.user.Class вместо lib.user.Class где нужно).
